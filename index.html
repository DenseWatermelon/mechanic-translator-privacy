Mechanic Translator

Privacy Policy

Last updated: March 9, 2026

Mechanic Translator stores vehicle, symptom, visit, and attached reference photo information locally on the device using on-device storage.

Data Collected

  • Car details entered by the user
  • Symptom notes entered by the user
  • Visit and repair history entered by the user
  • Reference photos selected by the user for symptoms or repair visits

Data Sharing

  • The app does not include account sync
  • The app does not transmit stored data to a backend service
  • The app does not include analytics or advertising SDKs

Device Permissions

  • Photo library access is requested only when the user chooses to attach photos to a symptom or repair visit

Data Retention

  • Data remains on the device until the user deletes entries or resets local data
